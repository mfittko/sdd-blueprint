# Slidev Presentation: From Idea to Production

This presentation showcases the spec-driven development approach used in the **Großmutters Rezepte** project, demonstrating how GitHub Copilot and Model Context Protocol (MCP) were used to build a production-grade web application.

## Quick Start

### Install Dependencies

```bash
npm install -D @slidev/cli @slidev/theme-default
```

### Run Presentation

```bash
npx slidev slides.md
```

This will start a local server at `http://localhost:3030` with hot reload enabled.

### Build for Production

```bash
npx slidev build slides.md
```

This generates a static site in `dist/` that can be deployed to any hosting service.

### Export to PDF

```bash
npx slidev export slides.md
```

Generates `slides-export.pdf` in the current directory.

## Customization

### Theme

The presentation uses the default Slidev theme. To customize:

```bash
npx slidev theme [theme-name]
```

### Layout Options

Each slide can use different layouts. Available layouts:

- `default` - Standard content slide
- `center` - Centered content
- `two-cols` - Two-column layout
- `image-right` - Content with right-aligned image
- `quote` - Large quote display

Example:
```markdown
---
layout: two-cols
---

# Left Column

::right::

# Right Column
```

### Code Highlighting

Specific lines can be highlighted:

```markdown
```typescript {1-5|7-12|14-20|all}
// Your code here
\```
```

The notation `{1-5|7-12}` creates progressive highlighting.

## Speaker Notes

Add speaker notes to any slide:

```markdown
---
# Slide Title

Content here

<!--
These are speaker notes that only you see
-->
```

View notes in presenter mode (press 'o' during presentation).

## Presenter Mode

During presentation, press:

- **Space** / **Arrow Keys** - Navigate slides
- **O** - Toggle overview mode
- **D** - Toggle dark mode
- **F** - Toggle fullscreen
- **G** - Go to specific slide
- **C** - Toggle camera view
- **P** - Toggle presenter mode (shows notes)
