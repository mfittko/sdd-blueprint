# Bug Report Template

**Issue Title:** `bug: <observable failure>`

## Summary
Clear description of the defect and the affected user flow.

## Environment
- Blueprint version / commit SHA
- Environment (local, CI, production)
- Browser or runtime details if applicable

## Steps to Reproduce
1. Step one
2. Step two
3. Step three

## Expected Result
Describe the correct behavior.

## Actual Result
Describe what actually happens, including error messages or screenshots.

## Impact Assessment
- Frequency (always, intermittent, rare)
- Severity (blocker, high, medium, low)
- Affected audiences

## Proposed Fix (Optional)
Notes about likely root cause or potential remediation path.

## Testing Notes
- Regression areas to cover
- New automated tests required (unit/integration/e2e)
- Manual QA steps after the fix
